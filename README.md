# E-Commerce Store

## Overview

This is a basic full-stack E-Commerce Store developed using HTML, CSS, JavaScript, Node.js, Express.js, and MySQL. The application allows users to browse products, view product details, add products to a shopping cart, register/login, and place orders.

## Features

* User Registration
* User Login
* Product Listings
* Product Details Page
* Shopping Cart
* Order Processing
* MySQL Database Integration
* Responsive User Interface

## Technologies Used

### Frontend

* HTML
* CSS
* JavaScript

### Backend

* Node.js
* Express.js

### Database

* MySQL

## Database Tables

* users
* products
* cart
* orders

## Installation

1.  repository link

```bash
link: https://github.com/Kalsoom-Gill/CodeAlpha_ecommerce-store.git
```

2. Install dependencies

```bash
npm install
```

3. Import the database.sql file into phpMyAdmin

4. Start the server in vscode terminal

```bash
node server.js
```

5. Open in browser

```text
links to check:
http://localhost/phpmyadmin
http://localhost:3000/register.html
http://localhost:3000/login.html
http://localhost:3000/index.html
http://localhost:3000/cart.html
http://localhost:3000/checkout.html
```
